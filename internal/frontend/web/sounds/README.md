# Notification sounds

Place two audio files here:

- `dm.mp3` — played when a direct message arrives
- `mention.mp3` — played when someone @mentions you in a room

Any browser-supported format works (mp3, ogg, wav). Short clips of 0.5–2 seconds are recommended.

If the files are missing the app falls back to synthesised tones generated by the Web Audio API.

Rebuild the binary after adding files so they are embedded:

    go build ./cmd/conduit
